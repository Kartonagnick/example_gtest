[![S]][H] actualize v0.0.1
==========================
Батник актуализирует исторические метки в файлах,  
т.е., удаляет из файлов упоминания `PRE`, и заменяет `[P]` на `[S]`  

Например, было:  

```
[![P]][H] **v0.0.3 (dev)**
...
# [2025-12-07][22:10:00] 004 Kartonagnick PRE
```

Стало:  

```
# [2025-12-07][22:10:00] 004 Kartonagnick    
...
[![S]][H] **v0.0.3 (dev)**
```

Батник используется при подготовке очередной официальной версии
