#4-dev-stash
============
Нужно добавить комплект вспомогательных скриптов для обслуживания репозитория.  

Для Windows:  
  - `[vbs]-replace.bat`  меняет исторические метки в файлах  
  - `calcRebase.bat`     определяет значение `master~N` для rebase  
  - `actualize.bat`      убирает PRE в исторических метках  
  - `docs.view.bat`      просмотр документации в броузере  
  - `7zrun.bat`          архивация репозитория  
  - `git-commits.bat`    коммит изменных/новых файлов  
  - `git-rebase.bat`     запуск интерактивного ребейза  
  - `git-branch.bat`     переключение на самую новую ветку  
  - `git-clean.bat`      очистка/сжатие git репозитория  
  - `git-date.bat`       установка/замены даты коммитов  

В дальнейшем, могут быть добавлены Linux-версии на bash.  
Но это будет уже в следующих задачах.
