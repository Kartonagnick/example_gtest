#2-dev-frame
============
В этой задаче осуществляются работы по созданию каркаса проекта.  
Схема классическая, 3х компонентная:  

```
                  +--------+
                  | deploy |
                  +--------+

             +------------------+    
          .--|    sample.lib    |--. 
         |   +------------------+   |
         |                          |
   +-----------+           +------------------+
   | tests.exe |           |    sample.exe    |
   +-----------+           +------------------+
         |
         |
      external
         |
         |
  +-------------+
  |  gtest.lib  |
  +-------------+
         |
  +-------------+
  |  gmock.lib  |
  +-------------+

```

deploy
------
Содержит скрипты сборки и деплоя проекта.  
<br/>


sample.lib
----------
Статическая библиотека.  
Содержит всю бизнес-логику плагина.  
<br/>


sample.exe
----------
Главная цель проекта: утилита, запускаемая из командной строки  
Реализует бизнес-логику руками `sample.lib`
<br/>


tests.exe
--------
Использует `gtest` и `gmock`.  
Осуществляет юнит-тестирование `sample.lib`  
<br/>
