#5-dev-stash
============
Нужно добавить комплект вспомогательных скриптов для обслуживания репозитория.  
Для Linux:  
 - `chg-data.sh`     меняет исторические метки в файлах  
 - `calcRebase.sh`   определяет значение `master~N` для rebase  
 - `actualize.sh`    убирает PRE в исторических метках  
 - `docs.view.sh`    просмотр документации в броузере  
 - `7zrun.sh`        архивация репозитория  
 - `git-commits.sh`  коммит изменных/новых файлов  
 - `git-rebase.sh`   запуск интерактивного ребейза  
 - `git-branch.sh`   переключение на самую новую ветку  
 - `git-clean.sh`    очистка/сжатие git репозитория  
 - `git-date.sh`     установка/замены даты коммитов  

В этой задаче делаем:
 - `git-commits.sh`  коммит изменных/новых файлов  
 - `calcRebase.sh`   определяет значение `master~N` для rebase  
 - `docs.view.sh`    просмотр документации в броузере
